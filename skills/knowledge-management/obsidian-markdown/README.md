# obsidian-markdown

> **Original Author:** [Kepano](https://github.com/kepano/obsidian-skills)

## Overview

Create and edit Obsidian Flavored Markdown with wikilinks, embeds, callouts, properties, and other Obsidian-specific syntax. Use when working with .md files in Obsidian, or when the user mentions wikilinks, callouts, frontmatter, tags, embeds, or Obsidian notes.

## Installation

Install this skill dynamically into any supported environment via the CLI:

```bash
npx skills@latest add surajgthakkar/agent-skills --skill obsidian-markdown
```

## Usage

Once installed, the agent will automatically parse the `SKILL.md` instructions when triggered. For complete instructions, arguments, and the raw prompt payload, refer to the [`SKILL.md`](./SKILL.md) file in this directory.
