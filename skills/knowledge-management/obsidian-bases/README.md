# obsidian-bases

> **Original Author:** [Kepano](https://github.com/kepano/obsidian-skills)

## Overview

Create and edit Obsidian Bases (.base files) with views, filters, formulas, and summaries. Use when working with .base files, creating database-like views of notes, or when the user mentions Bases, table views, card views, filters, or formulas in Obsidian.

## Installation

Install this skill dynamically into any supported environment via the CLI:

```bash
npx skills@latest add surajgthakkar/agent-skills --skill obsidian-bases
```

## Usage

Once installed, the agent will automatically parse the `SKILL.md` instructions when triggered. For complete instructions, arguments, and the raw prompt payload, refer to the [`SKILL.md`](./SKILL.md) file in this directory.
