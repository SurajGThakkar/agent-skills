# grilling

> **Original Author:** [Matt Pocock](https://github.com/mattpocock/skills)

## Overview

Grill the user relentlessly about a plan, decision, or idea. Use when the user wants to stress-test their thinking, or uses any 'grill' trigger phrases.

## Installation

Install this skill dynamically into any supported environment via the CLI:

```bash
npx skills@latest add surajgthakkar/agent-skills --skill grilling
```

## Usage

Once installed, the agent will automatically parse the `SKILL.md` instructions when triggered. For complete instructions, arguments, and the raw prompt payload, refer to the [`SKILL.md`](./SKILL.md) file in this directory.
